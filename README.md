## 🚀 How to Run

### Option 1: Using VS Code Live Server Extension
1. Install the "Live Server" extension in VS Code
2. Right-click on `index.html`
3. Select "Open with Live Server"
4. Website opens in your browser automatically

### Option 2: Using Python HTTP Server
```bash
cd gaurav-professional-portfolio
python -m http.server 8080
```
Then open: `http://localhost:8080`

### Option 3: Direct File Open
Simply double-click `index.html` to open in your browser (some features may be limited)

## 🌐 GitHub Pages Hosting

To host on GitHub Pages:

1. Create a new repository on GitHub
2. Upload all files (index.html, style.css, script.js)
3. Go to Settings → Pages
4. Select "Deploy from branch" → "main"
5. Your site will be live at: `https://username.github.io/repo-name`

## 🎨 Customization

### Colors
Edit CSS variables in `style.css`:
```css
:root {
    --primary-color: #00d4ff;      /* Neon blue */
    --secondary-color: #0066ff;    /* Blue */
    --accent-color: #00ffff;       /* Cyan */
    --bg-dark: #0a0e27;           /* Dark navy */
}
```

### Typing Effect Texts
Edit in `script.js`:
```javascript
const config = {
    titles: [
        "3D Artist",
        "Motion Graphic Designer",
        "Graphic Designer",
        "Video Editor",
        "VFX Compositor"
    ]
};
```

### Skills & Percentages
Edit skill bars in `index.html`:
```html
<div class="skill-progress" data-progress="95"></div>
```
