# Adobe Icon Set Integration Guide

## 📁 Icon Files Needed

Copy the following PNG files from the Adobe Icon Set 2022 RAR archive to the `icons/` folder:

```
icons/
├── Illustrator.png
├── Photoshop.png
├── InDesign.png
├── coreldraw-2019-seeklogo.png
├── Premiere Pro.png
├── After Effects.png
├── Audition.png
├── Animate.png
├── autodesk-maya-seeklogo.png
├── adobe substance painter.png
├── icons8-zbrush-480.png
├── icons8-nuke-480.png
└── Silhouette.png
```

## 🛠️ How to Extract

1. Download: `Adobe Icon Set 2022  Figma Icon (Community).rar`
2. Use WinRAR or 7-Zip to extract
3. Navigate to: `Adobe Icon Set 2022  Figma Icon (Community)/`
4. Copy all `.png` files to the `gaurav-professional-portfolio/icons/` folder

## ✅ Verification

Once copied, all tool icons in the Skills section will display properly!
